# 差距分析与优化 Backlog

## 总体评估

New API 在路由/容错/监控方面已有相当完善的基础。核心路由机制（优先级+加权随机+亲和性+自动熔断+跨组重试）已经支撑了生产级负载。主要缺失集中在**动态决策**和**标准化可观测性**领域。

## 差距分析

### 路由 — 现有能力很强，但全是静态配置

| 能力 | 状态 | 备注 |
|------|------|------|
| 静态优先级路由 | ✅ | Priority + Weight 静态配置，效果好 |
| 自动降级/熔断 | ✅ | 关键词/状态码匹配，自动禁用+恢复 |
| 会话亲和性 | ✅ | HybridCache 实现，支持多规则匹配 |
| 跨组降级 | ✅ | Auto group 用户逐组尝试 |
| 多 Key 负载分担 | ✅ | Random/Polling |
| **延迟感知路由** | ❌ | ResponseTime 已采集，但选择时没有使用 |
| **成本感知路由** | ❌ | 不同渠道价格不同，但选择时未考虑成本 |
| **动态权重调整** | ❌ | 权重只靠管理员手动配 |
| **滑动窗口熔断** | ❌ | 只有一次失败就禁用，无半开/错误率窗口 |
| **主动健康检查** | ❌ | 测试是定时的，不是实时的 |

### 监控 — 基础数据很全，但暴露方式有限

| 能力 | 状态 | 备注 |
|------|------|------|
| 请求日志 | ✅ | LOG_DB 完整记录每次调用的元数据 |
| 性能指标聚合 | ✅ | perf_metrics 表，含延迟/成功率/TTFT |
| 用量统计 | ✅ | quota_data + 统计 API |
| 系统资源监控 | ✅ | CPU/内存/磁盘阈值检查 |
| Pyroscope 集成 | ✅ | 可选持续性能分析 |
| **Prometheus 端点** | ❌ | 无法被标准监控栈采集 |
| **结构化日志 (JSON)** | ❌ | 无法被 Logstash/Loki 高效采集 |
| **延迟百分位 (P95/P99)** | ❌ | 只有平均值，看不到尾延迟 |
| **请求/响应体大小** | ❌ | 不跟踪 |
| **按状态码细分错误率** | ❌ | 只分成功/失败 |
| **渠道级延迟时间序列** | ❌ | 只有 `channel.ResponseTime` 一个值 |
| **健康检查端点** | ❌ | `/api/status` 过于简单 |
| **分布式追踪** | ❌ | 只靠 request_id 关联 |

### 告警 — 几乎空白

| 场景 | 当前 | 需要 |
|------|------|------|
| 整体成功率下降 | 无 | 5 分钟成功率 < 95% → P1 |
| 某模型全部渠道不可用 | 用户 503 | 全渠道连续失败 → P1 |
| 渠道超时率过高 | 自动禁用可能无通知 | 超时率 > 20% → P2 |
| 日成本超预算 | 无 | 日成本 > 阈值 → P2 |
| 渠道余额不足 | 无 | 余额 < 阈值 → P2 |

## 优化 Backlog

### 可配置项（不需要改代码）

| 序号 | 项目 | 操作 | 预期效果 |
|------|------|------|----------|
| C-01 | 启用 `RETRY_TIMES > 0` | 设置环境变量 | 故障时自动重试降级渠道 |
| C-02 | 启用 `CHANNEL_TEST_ENABLED` + 调频率 | 设置环境变量 | 定时检测渠道健康 |
| C-03 | 配置 `AutomaticDisableStatusCodeRanges` | 后台设置页面 | 更精确的自动禁用规则 |
| C-04 | 配置 Channel Affinity 规则 | 设置页面添加 rules | 更好的会话级路由 |
| C-05 | 配 `AutomaticEnableChannelEnabled` | 环境变量或后台 | 渠道恢复后自动启用 |
| C-06 | 为关键模型配 2+ 渠道不同 Priority | 渠道管理创建 | 多渠道冗余 |
| C-07 | 为多 Key 供应商启用 Multi-Key 模式 | 渠道编辑启用 | 单渠道多条 Key 负载分担 |

### 短期二开项（1-3 天，有明确实现路径）

| S-02 | **Prometheus /metrics 端点** | 暴露 Go runtime + 请求量/成功率/延迟直方图 | 新增 `controller/metrics.go` + `router` 注册 | 1.5d |
